🔧 Backend

  • Framework: NestJS
  • Language: TypeScript
  • Audio: yt-dlp, youtube-dl-exec
  • Storage: JSON + File System
  • Validation: class-validator

🎨 Frontend

  • Framework: Next.js 14+
  • Language: TypeScript
  • Styling: Tailwind CSS
  • State: Zustand
  • Audio: HTML5 Audio API

📊 DevOps

  • Containers: Docker
  • Metrics: Prometheus
  • Visualization: Grafana
  • Package: pnpm
  • Process: PM2

📚 Documentación de la API

🔄 Translation API

POST /api/translation/translate

Traduce texto entre wayuu y español

{
  "text": "wayuu",
  "direction": "wayuu-to-spanish"
}
GET /api/translation/stats

Obtiene estadísticas de traducción

GET /api/datasets/info

Información del dataset wayuu

📹 YouTube Ingestion API

POST /api/youtube-ingestion/ingest

Procesa un video de YouTube

{
  "url": "https://youtube.com/watch?v=ID"
}
GET /api/youtube-ingestion/status

Estado de videos procesados

POST /api/youtube-ingestion/process-pending

Procesa videos pendientes
